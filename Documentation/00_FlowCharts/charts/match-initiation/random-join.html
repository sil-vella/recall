Random join

Flow from lobby (Join Random Game) to backend and through the coordinator until initial match state is built. No invites; room has is_random_join: true, no accepted_players. Start is triggered by delay timer or when room reaches max players; coordinator fetches comp players from the Flask API. Contrast: Create match uses invites and accepted_players; effective max = maxSize − comp count; coordinator uses the accepted comp list.
