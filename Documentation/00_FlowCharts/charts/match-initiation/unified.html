Match initiation – all flows (unified)

Merge points: (1) clearAllGameStateBeforeNewGame – first in Practice and Create; inside joinRandomGame/joinRoom for Random and Join when not already in room. (2) ensureWebSocketReady – widget then helper for Random/Join; after clear and setTransportMode for Create; Practice uses in-app bridge only. (3) Navigate to /dutch/game-play – all paths converge here (Practice direct; Random new room via room_creation 300ms delay, existing room via user tap + setCurrentGameSync; Create/Joint via setCurrentGameSync after state ready). (4) Game play screen – same Start Match and round logic for all modes.
