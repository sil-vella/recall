Create tournament plan

From lobby Tournaments tab: existing = only public. Create new: auto private, select format (F1, F2, F3), invite players. User search filters out creator (creator auto-added/accepted). Number of invites required depends on format. Python backend creates tournament in DB (see prev commits). Invitees accept notification; when required number accepted, auto notification to all HUMAN players to join tournament. Then wire into existing match logic (add to room etc); game state gets tournament-specific keys in game map: tournament name, format, players, etc.
